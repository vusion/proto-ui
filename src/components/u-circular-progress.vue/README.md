# UCircularProgress 圆型进度条

## 示例
### 基本形式

``` html
<u-circular-progress :percent="36"></u-circular-progress>
```

## API
### Props/Attrs

| Prop/Attr | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| percent | Number | `0` | 百分比 |

### Slots

#### (default)

插入文本或 HTML。
